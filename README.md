# StudySpot

A community-driven iOS app for discovering and reviewing study spots around campus. Users can browse spots on a map or list, check real-time open/closed status, see busyness reports, and leave reviews with ratings — all updating live across devices.

## Technologies Used

- **Swift & SwiftUI** — built with modern SwiftUI patterns taught in Prof. John Gallaugher's BCSwift course (Boston College, Spring 2026)
- **Firebase Firestore** — real-time NoSQL database with live snapshot listeners for spots and reviews
- **Firebase Storage** — photo uploads with download URLs stored in Firestore documents
- **MapKit** — interactive map with custom annotations and spot discovery
- **CoreLocation** — auto-fills coordinates when adding a new spot
- **MVVM Architecture** — `@Observable` ViewModels managing all data and business logic
- **Xcode 26 / iOS 26** — built and tested on the latest Apple developer toolchain
